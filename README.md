# 医院管理系统 java933

本项目是基于小程序的医院管理系统，后端框架采用了Springboot技术，前端使用了VUE框架。系统功能完善，涵盖了医院日常运营所需的管理功能。

## 技术栈

- 后端：Springboot
- 前端：VUE

## 功能模块

### 系统管理员功能

- 个人中心
- 用户管理
- 医生管理
- 排班信息管理
- 医生信息管理
- 科室信息管理
- 科室预约管理
- 病历信息管理
- 体检报告管理
- 系统管理

### 医生功能

- 管理排班
- 处理预约信息
- 管理体检信息
- 管理病历信息

### 用户功能

- 注册与登录
- 预约服务

## 系统角色

- 系统管理员
- 医生
- 患者（用户）

## 运行环境

- 后端：Java运行环境
- 前端：适用于现代Web浏览器
